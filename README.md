## Getting Started

### Prerequisites

Make sure you have the following installed:
- Node.js
- npm
- MongoDB
- Mongoose
- chai
- mocha
- chai-http

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/raj992722/productApi.git

2. Install dependencies

    ```bash
    
   cd e-commerce-api
   npm install

3. Running the application

    ```bash
    npm start
